⚙ Gothic Remake — Lock Solver

Set the starting pin positions, describe how each mechanism pushes the others, then hit Solve. The goal is to get every pin into the dashed center hole.

1 · Starting positions

Click a hole to place each mechanism's pin where it currently sits in the game.

2 · Interaction matrix

Read each row as: "When I move this mechanism, the column mechanism reacts by…" Click a cell to cycle None → Same → Opposite. The dashed diagonal is how a mechanism moves itself (normally Same).

Same direction = it moves the way you pushed.   Opposite direction = it moves the other way.   Map the game's mirror / counter to whichever of these you observe.

3 · Solve

4 · Auto-play (optional)

Turns the solution into a list of key presses. Selection starts on the last mechanism; W/S move the selection between mechanisms, A/D slide the selected one. After you Solve, click Copy key plan, then run autoplay.py — it counts down and presses the keys for you (switch to the game during the countdown).
