1 · Starting positions

Click a hole to place each mechanism's pin where it currently sits in the game.

2 · Interaction matrix

Read each row as: "When I move this mechanism, the column mechanism reacts by…" Click a cell to cycle None → Same → Opposite. The crossed-out diagonal is a mechanism on itself — fixed, since moving it always moves itself.

Same direction = it moves the way you pushed.   Opposite direction = it moves the other way.   Map the game's mirror / counter to whichever of these you observe.

4 · Auto-play (optional)

Turns the solution into a list of key presses. Selection starts on the last mechanism; W/S move the selection between mechanisms, A/D slide the selected one. In the desktop app you can press Execute below to play it automatically. On the website, click Copy key plan and run the autoplayer.

Desktop app detected — Execute focuses the game, presses the reset key, waits, then plays the moves. Press Stop or Esc to abort.
